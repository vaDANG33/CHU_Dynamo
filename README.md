# CHU_Dynamo

Scripts Dynamo pour Revit - Collection d'outils et de scripts pour automatiser les tâches dans Revit via Dynamo.

## 📋 Description

Ce projet contient une série de scripts Dynamo conçus pour faciliter et automatiser les workflows dans Revit au sein de l'environnement CHU.

## 🎯 Objectifs

- Automatiser les tâches répétitives dans Revit
- Améliorer la productivité des utilisateurs
- Fournir des outils réutilisables et maintenables
- Documenter les bonnes pratiques Dynamo
